COMPONENT HydraulicPressureSource ps1
COMPONENT HydraulicPressureSource ps2

COMPONENT MechanicForceTransformer f
COMPONENT SignalSource fs

SET ps1 P 100000
SET ps2 P 140000
SET fs Value 0.0
SET cyl Area1 0.001
SET cyl Area2 0.001
SET cyl Bp 0.0
SET cyl Mass 100
SET cyl Stroke 1.0
SET cyl Kl 0.0

CONNECT ps2 P1 cyl P2

CONNECT fs out f in
